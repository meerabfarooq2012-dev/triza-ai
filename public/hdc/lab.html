2️⃣ XOR Binding Combine two concepts

XOR is the CPU's fastest operation. When we XOR two vectors, we get a new vector that is the "combination" of both. This is the heart of HDC.

Result: A XOR B
XOR result first part: Make Vector A and B first
💡 XOR magic: A XOR B XOR A = B (you get B back!). This is why HDC memory is "holographic" — everything can be recovered.

3️⃣ Hamming Distance How many bits differ?

Hamming Distance = count of bits that differ between two vectors. Less distance = more similar. This is how HDC recognizes things.

Result: Make both vectors first
💡 For 256 bits, two RANDOM vectors have distance ≈ 128 (50%). If distance is less than 100, they are considered "similar".

4️⃣ Memory & Recognition Real AI!

Now the real work. Store some words in memory. Then give a new word — the AI will tell you which stored word matches the most.

💡 Each word gets its own random 256-bit "name". The same word always makes the same vector (deterministic). Test it below.
